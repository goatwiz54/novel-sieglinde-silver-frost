# Noble Naming System (Canonical)

---

## 概要 / Overview

本ドキュメントは、物語世界における**個人名・家名・領地・国家役職**を  
**一貫した法的・社会的意味**として表記するための命名体系を定義する。

この体系では、名前の並び順そのものが**権限の階層**を表す。

```
個人 ＜ 家・私的統治権 ＜ 国家からの役職（公権）
```

---

## 基本構造 / Basic Structure

```
[個人名] ・ [識別子] ・ [地名]
```

※ 日本語表記上の「・」は区切り記号であり、  
　世界内言語における意味は持たない。

---

## 識別子一覧 / Identifiers

### 1. da（ダ）

**意味 / Meaning**

- 出身・由来・移住元を示す暫定的識別子
    

**定義 / Definition**

> da は、ある人物がどの土地から来たかを示す法的補助表現であり、  
> 血統・家名・統治権を一切示さない。

**使用条件 / Usage**

- 移住直後
    
- 外来者
    
- 裁判記録・住民登録・通行証など
    

**特徴 / Notes**

- 一時的
    
- 定住が認められると外れる
    

**例 / Example**

```
カイン・ダ・レンドル
(Kain da Rendor)
```

---

### 2. （識別子なし）＝ 家名

**意味 / Meaning**

- 定着した姓・血統名
    

**定義 / Definition**

> 家名は、その人物がどの家系・家族単位に属するかを示す。  
> 出身地由来であっても、定着後は血統名として扱われる。

**例 / Example**

```
カイン・レンドル
アロヴェルト・レンドル
```

---

### 3. rom（ロム）

**意味 / Meaning**

- 私的統治権・領有権
    

**定義 / Definition**

> rom は、王権または国家により土地の統治を認められた者に付される、  
> 現在形の私的統治権を示す標識である。

**使用条件 / Usage**

- 土地を実際に治めている者のみ
    
- 領主・土地持ち
    

**ルール / Rules**

- 土地を失えば即座に外れる
    
- 世襲可能
    
- 出身・血統とは無関係
    

**例 / Example**

```
アロヴェルト・ロム・レンドル
(Arovert rom Rendor)
```

---

### 4. roz（ロズ）

**意味 / Meaning**

- 国家から任命された役職・公的権限
    

**定義 / Definition**

> roz は、王・国家・公的機関から委任された役職に就いていることを示す  
> 公権的標識であり、私領の有無とは無関係である。

**使用条件 / Usage**

- 総督・代官・司令官・監督官など
    
- 任期制
    

**ルール / Rules**

- 免職・退任で外れる
    
- 相続不可
    
- rom より上位の権威
    

**例 / Example**

```
アロヴェルト・ロズ・ライル
(Arovert roz Lyle)
```

---

## 複合表記 / Combined Forms

### 私領の恒久統合（＝）

**意味 / Meaning**

- 二つ以上の私領を不可分の家領として統合したことを示す
    

**条件 / Conditions**

- 王権の裁可
    
- 恒久統合
    

**例 / Example**

```
ロム・レンドル＝ウィンダス
```

---

### 完全表記（正式文書用）

```
アロヴェルト・ロム・レンドル＝ウィンダス・ロズ・ライル
```

**意味 / Meaning**

- レンドル＝ウィンダス統合私領の領主
    
- 同時にライル領の国家任命総督
    

※ 叙爵文・王国台帳・条約文・裁判記録などで使用

---

## 権限階層と表記順 / Authority Order

```
個人名
↓
家名・私的統治（rom）
↓
国家からの役職（roz）
```

**原則**

- 弱い権限から強い権限へ記述
    
- 私権 → 公権

|国家役職|アロヴェルト・ロズ・ライル|
|領主＋役職|アロヴェルト・ロム・レンドル・ロズ・ライル|

---

## 正史用 定義文 / Canonical Definition

> **da は出身を示す暫定的法的識別子、  
> rom は現在の私的統治権を示す標識、  
> roz は国家から委任された公的役職を示す標識である。**

---

## 備考 / Notes

- 本体系は物語本文・設定資料・年表・裁判記録すべてに適用される。
    
- 名前の変化は、身分・権限・人生の変遷を直接的に表現する。
