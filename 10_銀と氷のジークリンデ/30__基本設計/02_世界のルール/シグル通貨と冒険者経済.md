## 受けられる依頼ランク

### ソロの場合

| 冒険者ランク | 受けられる依頼 |
|------------|--------------|
| 黒曜石 | 黒曜石のみ |
| 石英 | 黒曜石・石英・青玉 |
| 青玉 | 石英・青玉・紅玉 |
| 紅玉 | 青玉・紅玉・翠玉 |
| 翠玉 | 紅玉・翠玉・白金晶 |
| 白金晶 | 翠玉・白金晶・ミスリル |
| ミスリル | 白金晶・ミスリル |

### PT編成の場合

| PT編成 | 受けられる依頼 |
|--------|--------------|
| 黒曜石ソロ | 黒曜石のみ |
| 黒曜石が含まれるPT | 黒曜石・石英・青玉（上限は青玉） |
| 石英×全員 | 黒曜石・石英・青玉 |
| 石英＋青玉混成 | 石英・青玉 |
| 青玉×全員 | 石英・青玉・紅玉 |
| 青玉＋紅玉混成 | 青玉・紅玉 |
| 紅玉×全員 | 青玉・紅玉・翠玉 |
| 翠玉×全員 | 紅玉・翠玉・白金晶 |
| 白金晶×全員 | 翠玉・白金晶・ミスリル |
| ミスリル×全員 | 白金晶・ミスリル |

> ルール：混成PTは最低ランクから2つ上まで。同ランク全員PTは上下1つずつ。
> 黒曜石が含まれると上限が青玉になるため、上位冒険者にとってデメリットになる。

## ランクポイントのルール

| 状況 | ランクポイント |
|------|--------------|
| 同ランク全員PT | 加算あり |
| 異ランク混成PT（通常） | 加算なし |
| 大規模討伐（大隊列等） | ランク混在でも加算あり |

### 異ランク混成PT（通常）のデメリット
- ランクポイントが貯まらない
- 報奨金に税金がかかる（手取りが下がる）
- 上位冒険者が積極的に低ランクと組む理由がない

### 大規模討伐が特例な理由
- 人数が必要な案件はランク混在を認めないと成立しない
- 国家・ギルドが認定した大規模依頼に限る
- アドラス討伐のような大隊列はこの特例に該当

### 備考
- バルザードとレンドルは師弟関係であり正式なPTではない
- レンドルとルテウスは同PT（ただしランク差あり・混成扱い）
- 大規模討伐参加によるランクポイント加算が、黒曜石級レンドルにとって重要な昇格機会となる

## 黒曜石級・月収モデル（ソロ〜4人パーティ）

### 前提条件
- 石狼討伐：1匹約2万円
