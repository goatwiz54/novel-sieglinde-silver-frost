# Elven Naming: Marriage & Children (Canonical)

---

## 概要 / Overview

本ドキュメントは、エルフ社会における**結婚後の名前の扱い**および  
**子の命名規則**を正史として定義するものである。

エルフの命名は、人間社会のような  
「家・姓・制度」を中心とせず、  
**関係性・共同体・育成**を中心に構成される。

---

## 基本思想 / Core Philosophy

> **エルフにとって名前とは、  
> 所有や制度を示すものではなく、  
> その存在がどこに連なり、  
> 誰と生きてきたかを語るものである。**

このため、結婚・役職・権力は  
原則として名前に刻まれない。

---

## 1. 結婚と名前 / Marriage & Names

### 原則 / Rule

- **結婚しても名前は変わらない**
    
- 改姓・家名統合は行わない
    
- 夫婦関係は名前ではなく文脈で示す
    

---

### 例 / Example

#### 結婚前

```
ゼイナリンデ・ルルザ・ホシル
ネオネス・トア・モートン
```

#### 結婚後（変更なし）

```
ゼイナリンデ・ルルザ・ホシル
ネオネス・トア・モートン
```

**補足**

- 「ネオネスの伴侶、ゼイナリンデ」など、  
    関係性は文章で表現される。
    

---

## 2. 子の命名規則 / Child Naming Rules

### 基本構造 / Basic Structure

```
[個人名]・[育った地]・[育ての親]
```

※ 親名は「血縁」ではなく  
**育成・導きの関係**を示す。

---

## 3. 具体例：ジークリンデの場合

### 前提 / Context

- 父：ネオネス・トア・モートン
    
- 母：ゼイナリンデ・ルルザ・ホシル
    
- 居住地：ブリグルド
    

---

### 基本名 / Common Name

```
ジークリンデ・ブリグルド・ネオネス
```

**意味**

- ジークリンデ：個人名
    
- ブリグルド：育った共同体
    
- ネオネス：育ての親
    

---

### 変化例 / Variations

#### 母に育てられた場合

```
ジークリンデ・ブリグルド・ゼイナリンデ
```

#### 共同体に育てられた場合

```
ジークリンデ・ブリグルド
```

---

## 4. 対外的・正式表記 / Formal External Name

外部者・儀礼・記録用途では  
最大情報を含む表記が用いられる。

```
ジークリンデ・アズ＝シルヴァン・ブリグルド・ネオネス
```

---

## 5. 人間社会との対比 / Contrast with Humans

|項目|人間|エルフ|
|---|---|---|
|結婚|家の統合|関係の追加|
|改姓|あり|なし|
|子の姓|家名継承|育成重視|
|役職|名前に含む|名前に含めない|

---

## 正史定義文 / Canonical Definition

> **エルフは結婚によって名を変えない。  
> 子は、育った地と育てた者の名によって呼ばれる。  
> 名は制度ではなく、関係を語るものである。**
