# 命名体系（正史）

---

## 1. 基本構造

個人の正式な名乗りは、以下の要素で構成される。

```
［個人名］・［家名］（＋爵位）・［カル］・［ロム］・［ロズ］
```

※ 実際の名乗りでは、冗長な要素は省略される。

---

## 2. 各要素の定義

### ■ 爵位
- 騎士爵／伯爵／公爵など
- **身分保障を内包する**ため、爵位保持者は自動的に貴族身分を持つ
- 個人爵位（騎士爵）と家爵位（伯爵・公爵）がある

例：
```
ファレン・ブレイズ騎士爵
```

---

### ■ カル（貴族身分登録）
- 国家が認めた「貴族身分」を示す識別語
- **爵位を持たない者が貴族であることを示すために使用**
- 爵位保持者は、原則としてカルを省略する

#### カルの種類
- 血統カル：爵位家の正統子として自動付与
- 登録カル：国家登録・納税によって取得

例：
```
レンドル・ブレイズ・カル
```

---

### ■ ロム（私的統治権）
- 国家が認めた私領統治権
- **カルまたは爵位を前提として付与される**

例：
```
ロム・ウィンダス
```

---

### ■ ロズ（国家役職）
- 国家・軍・行政における公式役職
- ロムと同列、または独立して付与される

例：
```
ロズ・ライル
```

---

## 3. 名乗りの実例

### 騎士爵で領と役職を持つ場合
```
レンドル・ブレイズ騎士爵・ロム・ウィンダス＝ジャール・ロズ・ライル
```

### 無爵位だが貴族身分で領を持つ場合
```
レンドル・ブレイズ・カル・ロム・ウィンダス＝ジャール
```
